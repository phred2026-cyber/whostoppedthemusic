# Who Stopped The Music

A client-side Minecraft mod for **Fabric 1.21.x** that ensures vanilla music always plays without silence gaps.

## What it does

- Eliminates the random silence between tracks (up to ~2.75 minutes vanilla)
- Prevents music from being cut short mid-song
- **Leaves vanilla music selection 100% intact** — cave music, biome music, boss music, and intentional silences (like the Pale Garden) all work as designed

## What it doesn't do

- Does NOT change which music plays where
- Does NOT add new music
- Does NOT affect sound effects, ambient sounds, or records

## Installation

Drop the `.jar` into your `mods/` folder. Client-side only — no server install needed.

## Compatibility

- Minecraft 1.21.1+
- Fabric Loader 0.15.0+
- Fabric API required
